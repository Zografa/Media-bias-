# Media-bias-
Картиране на Българската Медийна Екосистема 2024-2025: Стратегически Анализ на Топ 50 Медии по Методологията Ad Fontes

Интерактивна уеб страница в стил **Ad Fontes** за българските медии (2025), базирана на доклада:

- `Проучване на български медии_ Точност и пристрастия (1).odt`

## Как да стартираш локално

Това е статичен сайт (HTML/CSS/JS).

```bash
python -m http.server 8000
```

След това отвори: `http://localhost:8000`

## Какво включва

- Scatter chart с 50 медии, по оси:
  - X: пристрастие (либерално → център → консервативно / про-руско)
  - Y: надеждност (0-64)
- Филтри по:
  - търсене
  - тип медия
  - диапазон на надеждност
- Детайлен панел при избор на медия

## Качване в GitHub така че да работи (GitHub Pages)

1. Commit/push файловете в `main` (или в branch и после merge).
2. В GitHub отвори: **Settings → Pages**.
3. В **Build and deployment** избери:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` и `/ (root)`
4. Запази и изчакай deploy (обикновено 1-2 минути).
5. Отвори получения URL за сайта (примерно `https://<username>.github.io/Media-bias-/`).

> Началната страница е `index.html`, затова ще се зареди директно след deploy.
